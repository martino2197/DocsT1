Es un servicio creado para poder cotizar y generar guías.


Requests API
Las peticiones requieren por seguridad Autenticarse por el esquema de autenticación Bearer, por lo que previamente se requiere obtener un access_token, el cual regresara un token de autenticación.

Headers
Todas las peticiones deben llevar los siguientes Headers :
\{
    Accept: "application/json",
    Authorization: "Bearer \{token\}"
\}
