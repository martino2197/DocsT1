Este endpoint devuelve el árbol completo de categorías asociadas a un canal de venta específico.
Proporciona un listado detallado de las categorías disponibles dentro del canal seleccionado.

El parámetro IDSalesChannel puede tener los siguientes valores:
- t1: T1 Comercios
- CS: ClaroShop
- SR: Sears
- SN: Sanborns
