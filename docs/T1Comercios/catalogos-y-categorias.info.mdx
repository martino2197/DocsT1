La API de Catálogos y Categorías proporciona funcionalidades para consultar y administrar información relacionada con marcas y categorías de productos en T1Comercios.

Esta API ofrece las siguientes funcionalidades:

- Consulta de Marcas: Permite obtener información sobre todas las marcas oficiales disponibles en el sistema.

- Consulta de Categorías: Permite obtener información detallada sobre las categorías de productos en un catálogo específico.

- Consulta de Coincidencias: Permite identificar coincidencias entre categorías y canales de ventas, incluyendo un match con el catálogo SAT para cumplir con la carta porte.

Esta API es útil para integrar y gestionar eficientemente la información de productos y categorías en un entorno de ventas multicanal. Asegúrate de proporcionar las credenciales adecuadas y cumplir con los requisitos de autorización para acceder a las funcionalidades ofrecidas.
