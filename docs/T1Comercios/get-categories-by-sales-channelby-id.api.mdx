El response proporciona información detallada de la categoría seleccionada, incluyendo atributos y variaciones que definen los productos asociados a la categoría.

- Atributos:
Los atributos son características únicas y específicas de la categoría que definen los productos dentro de la misma. Estos atributos se encuentran dentro del nodo "property" en el objeto "data". Cada atributo tiene un ID único, nombre, longitud máxima y mínima, requisito de obligatoriedad, tipo de atributo y una lista de posibles valores permitidos.

 Ejemplo de atributo:
 - ID: 10176
 - Nombre: Altura de la Bota
- Longitud Máxima: 500
- Longitud Mínima: 1
- Requerido: false
- Tipo: list
- Valores Permitidos: tobillo

- Variaciones:
Las variaciones representan las diferentes versiones o opciones que pueden tener los productos dentro de la categoría. Por ejemplo, las variaciones podrían ser la talla, el color, el material, etc. Cada variación tiene su propio conjunto de atributos que definen las características específicas de esa variación. En este ejemplo, se incluye una variación de "Talla" con un único valor de "Extra Grande".

 Este response proporciona una visión detallada de la estructura y características de la categoría, incluyendo los atributos que definen los productos y las opciones de variación disponibles.
