<MethodEndpoint
  method={"get"}
  path={"/cm/v2/sales_channel/category/{category_id}/matches"}
>
  
</MethodEndpoint>



Este endpoint permite obtener coincidencias de una categoría T1 con los canales de ventas disponibles, incluyendo un match con el catálogo SAT para cumplir con la carta porte. 

Al enviar una solicitud a este endpoint con el ID de la categoría, se obtendrán las coincidencias de esa categoría con los diferentes canales de ventas, incluyendo detalles del match con el catálogo SAT.

Las coincidencias representan la correspondencia entre una categoría específica y los canales de ventas asociados, lo cual es útil para identificar dónde se pueden vender los productos de esa categoría y cumplir con los requisitos de la carta porte.

Nota: Es importante proporcionar el ID correcto de la categoría para obtener las coincidencias precisas con los canales de ventas, incluyendo el match con el catálogo SAT.
